### Setup

#### Enable MQTT broker
Go to the runtime dedicated to serving as an MQTT broker at localhost:48030
<img width="755" height="936" alt="image" src="https://github.com/user-attachments/assets/0ad049df-b571-4cfb-883f-7a4aec5650e3" />

#### Enable Environment Variables
Login to Primary Intelligence Hub at localhost:48000 and enable Settings >> Environment Variables.
Then restart the container for it to take effect.

<img width="689" height="986" alt="image" src="https://github.com/user-attachments/assets/0341730b-4b2f-4de0-afa7-11f91f46e3c4" />
