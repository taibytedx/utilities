# Generate SAML SP XML Metadata
## Pipeline reads in /appData/intelligencehub-identityproviders.json and generates SAML SP XML Metadata

### Usage

Import the intelligencehub-configuration.json. This will read in ../appData/intelligencehub-identityproviders.json and output an XML file to ../FileOutput/intelligencehub-SAML-XML-Metadata.xml
